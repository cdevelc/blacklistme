What is the Purpose of BlackListMe?

BlackListMe is a web service that allows end users to announce globally that an email address is no longer used and email from that address should be treated as spam.

Eliminate your old email address from spam

Provide Email Address

It only takes a few seconds. Tell us an email address that you no longer use, never send from, and don't want used by spammers.

Verify Ownership

We will send a verification message to that address. You simply click on the message URL and we will take it from there.

Rest Easy

Mail servers and processors that access our API will know immediately that your old address should never appear in the From: field of an email. Spammers using that address will be frustrated.

Use our lists and yours to filter email

Known Spammy Addresses
